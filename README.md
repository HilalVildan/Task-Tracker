# React Task Tracker

Create Task Tracker using React

<hr />

### Installation Options

```
$ git clone https://github.com/HilalVildan/Task-Tracker.git
```

```
$ npm install
```

```
$ npm start
```

### Or

```
$ git clone https://github.com/HilalVildan/Task-Tracker.git
```

```
$ yarn
```

```
$ yarn start
```
